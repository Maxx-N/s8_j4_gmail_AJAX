**__POUR LANCER L'APP EN LOCAL :__**

    1) Dans votre terminal, tapez les commandes : 

            * git clone https://github.com/Maxx-N/s8_j4_gmail_AJAX.git

            * s8_j4_gmail_AJAX

            * rails db:create

            * rails db:migrate

            * rails db:seed

            * rails server


    2) Dans votre navigateur, saisir l'URL : 

        http://localhost:3000/
