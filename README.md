**__POUR LANCER L'APP EN LOCAL :__**

    __1) Dans votre terminal, tapez les commandes :__

            git clone https://github.com/Maxx-N/s8_j4_gmail_AJAX.git

            s8_j4_gmail_AJAX

            rails db:create

            rails db:migrate

            rails db:seed

            rails server


    __2) Dans votre navigateur, saisir l'URL :__

        http://localhost:3000/
